Что было реализовано в данном проекте?

1. Используется backend, динамически генерирующий по крайней мере три различные страницы (список районов, список школ, страница каждой школы, страница каждого района) 2. Сайт автоматически обращается к каким-то API и использует возвращенную ими информацию (api data.mos.ru) 3. Сделана визуализация с помощью Plolty на странице со списком школ 4. На frontend реализована интерактивность с помощью JavaScript - нажатие кнопки "результаты" обновляет страницу без перезагрузки. 5. Используются AJAX запросы 6. Таблицы оформлены с помощью CSS
